## Convert an AGS file to GeoPackage

In QGIS: select the ags-tools processing plugin AGS2DB...

## Edit geology codes

In QGIS: select the GEOL layer - view attribute table - toggle editing, change/add data, save

## Create an exploratory hole plan

In QGIS: select the layers required on the map. Select File-Layout Manager....

## View summary plots such as SPT vs depth

In QGIS: select the ags-tools processing plugin - Summary

## Plot and analyse data

In QGIS: select the ags-tools processing plugin - DB2CSV or AGS2CSV

## Create graphical exploratory hole logs

!!! tip
    GeoPackage file must be loaded in QGIS

In QGIS: select OpenLog. Follow the guidance here (link to OpenLog plugin guidance)

## Filter data by selecting exploratory holes on a map

!!! tip
    Relationships in QGIS are required.

In QGSI: ensure Relationships are set and Select By Relationship plugin is installed and selected. See guidance here ()
