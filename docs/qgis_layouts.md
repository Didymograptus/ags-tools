Layouts are a built-in feature of QGIS they provide the means to create a report-quality output showing the map and/or other items such as tables, DataPlotly Graphs and images.

Layouts use a templating system so that a template such as a standard drawing frame can be used for may different outputs.

For Geotechnical outputs that would typically be found in reports, the types of outputs would be A4 portrait site location and parameter vs depth/elevation graphs and A3+ landscape or portrait exploratory hole location plans.

The 'QGIS National Highways package' comes with a few templates and it is expected that these will be adopted and extended by National Highways staff.

It should be noted that whilst the Layouts are accessed from within QGIS, the items such as images can be from any application or source. images could include photographs, charts from PowerBI or Excel etc. etc. The primary reason for using QGIS layouts is to ensure consistency or report-quality output.

It is recommended that the reader familiarises themselves with Layouts by reading the QGIS documentation [here](https://docs.qgis.org/3.44/en/docs/training_manual/map_composer/map_composer.html) or watch a [video here](https://youtu.be/ojRrIZ0M8o4?si=d9_sPjciCH6FUjWf)
