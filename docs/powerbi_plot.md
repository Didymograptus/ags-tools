## Opening the Power BI Template

Open the supplied **AGS Data Visualisation Dashboard Power BI template** file.

The template file will usually have the extension:

*.pbit

When the template opens, Power BI will ask for a parameter called **FolderPath**.

This is where you tell Power BI where the converted AGS CSV files are stored.

### FolderPath

In the **FolderPath** box, enter the full path to the folder containing the CSV files.

!!! example
    C:\Projects\Example Site\AGS CSV Output\

    !!! note
        You should enter the folder path only. Do not select an individual CSV file.

After entering the folder path:

1. Click **Load**.
2. Power BI will read the CSV files in the folder.
3. The dashboard will refresh.
4. The report pages will populate with the imported AGS data.

Depending on the size of the AGS dataset, the first load may take a few moments.
